# 🎓 GradPlots – Student Dashboard Landing Page UI

A modern and interactive **student dashboard landing page UI** designed in Figma for helping students manage their academic and college planning journey.
This project focuses on creating a structured, user-friendly, and visually engaging dashboard experience for students exploring colleges, mentorship, essays, and career planning resources.
[Prototype](https://www.figma.com/proto/fg2Rde200ww88RcV1qcJ9i/gradie-dash?node-id=2-4&t=r5edFxPVtIyJwVkw-1)
---

# ✨ Features

- 📄 Resume Browsing Section
- 🏫 Saved Colleges Dashboard
- 🎯 Personalized Student Journey
- 👨‍🏫 Mentor Support Access
- 🤝 Peer Connect Feature
- ✍️ Essay Support Section
- 🧠 Psychometric Test Access
- ✅ Checklist Tracking
- 📈 Profile Completion Tracker
- 🧭 Clean Dashboard Navigation

# 🎯 Project Goal

The goal of this project was to:
- Improve dashboard usability
- Create a visually organized student portal
- Build an engaging academic planning experience
- Design scalable and reusable UI components
- Enhance accessibility and navigation flow
